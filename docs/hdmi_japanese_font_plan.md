# HDMI 日本語（ひらがな・カタカナ）フォント設計書

本ドキュメントは、HDMI表示モジュールにおける日本語（ひらがな・カタカナ）表示機能の設計およびマッピング対応表を定義します。

## 背景と目的
0〜32番の文字コード領域は制御文字および標準スペース用として予約し、日本語フォントを配置しません。
標準ASCIIコード（33〜127）の直後である「128番」から、ひらがな・カタカナ・各種記号を連続したインデックスでマッピングします。
128〜255の128スロットのうち、空いている最後の10スロット（246〜255）を使用して、ひらがなの主要な濁音「が〜ぞ」をマッピングします。その他の濁音・半濁音は基本文字＋濁点・半濁点記号（゛, ゜）の結合文字として表現します。

## 制御文字の定義と3文字表記
文字コード表の視認性向上のため、制御文字およびスペースは `[]` 内に標準的な3文字略称で表示します。

- **007**: `BEL` (Bell)
- **008**: `BCK` (Backspace)
- **009**: `TAB` (Horizontal Tab)
- **010**: `RTN` (Line Feed / Return)
- **011**: `VT ` (Vertical Tab)
- **012**: `FF ` (Form Feed)
- **013**: `CR ` (Carriage Return)
- **014**: `SO ` (Shift Out)
- **015**: `SI ` (Shift In)
- **016**: `DLE` (Data Link Escape)

## ゲームボーイ向けゲーム開発に必要な制御文字の検討
ゲームボーイ（GBC）のようなリソースが制限された環境におけるテキストエンジン・RPGメッセージ表示システムでは、標準ASCII制御文字のほか、以下のような独自の制御文字コード（エスケープコード）を文字列バッファ内に埋め込んで処理することが一般的かつ非常に有効です。

1. **改行・制御系**
   - **`RTN` (改行)**: カーソルを次の行の先頭へ進める。画面が最大行数に達している場合は、後述の `WAI` 処理と組み合わせて自動スクロールする。
   - **`BCK` (バックスペース)**: 名前入力画面などで直前の文字を消去し、カーソルを1文字分戻す。
   - **`SPC` (スペース)**: 空白。

2. **テキスト送り制御（RPG等に必須）**
   - **`WAI` / `PRM` (ウェイト/プロンプト)**: メッセージウィンドウが一杯になった、または段落の終わりで描画を一時停止し、ユーザーの「Aボタン」入力を待つ。このとき画面右下に点滅する矢印マーク（▼）などを表示して入力を促す。
   - **`DLY` (時間遅延)**: メッセージ演出（間を持たせるなど）のために、一定のフレーム数（例: 30フレーム等）の間、文字描画を一時停止する。

3. **変数インジェクション・置換系**
   - **`NAM` (プレイヤー名)**: プレイヤー自身が決めた主人公の名前などのテキストバッファから動的に文字列を展開・挿入する。
   - **`RIV` (ライバル名)**: 同様にライバルキャラ名などを展開する。
   - **`ITM` (アイテム名)**: 入手したアイテムなどの動的な文字列を展開する。

4. **画面・音響演出制御**
   - **`SPD` (表示速度設定)**: 文字が1文字ずつ「カタカタ」と流れる表示速度（ウェイトフレーム数）を高速・低速・瞬時に変更する。
   - **`CLR` / `PAL` (テキスト色・パレット変更)**: 特定の単語（例: 重要アイテム、モンスター名）の色を赤や青に動的に切り替える。
   - **`SND` / `SFX` (効果音トリガー)**: メッセージ表示中や特定のセリフで特定の効果音（ピピッ、ジャーンなど）を再生するシグナル。


## 文字コードマッピング対応表 (0〜255)

### 1. 制御文字・スペース領域 (0〜32)
標準制御文字およびスペース領域として予約します。

### 2. ASCII 印刷可能文字領域 (33〜127)
標準ASCIIコード表に準拠します。
- 34: `"`, 126: `~`, 127: DEL

### 3. 日本語ひらがな領域 (128〜182)
- 128〜132: あ, い, う, え, お
- 133〜137: か, き, く, け, こ
- 138〜142: さ, し, す, せ, そ
- 143〜147: た, ち, つ, て, と
- 148〜152: な, に, ぬ, ね, の
- 153〜157: は, ひ, フ, へ, ほ
- 158〜162: ま, み, む, め, も
- 163〜165: や, ゆ, よ
- 166〜170: ら, り, る, れ, ろ
- 171〜173: わ, を, ん
- 174〜178: ぁ, ぃ, ぅ, ぇ, ぉ
- 179: っ
- 180〜182: ゃ, ゅ, ょ

### 4. 日本語カタカナ領域 (183〜237)

- 193〜197: サ, シ, ス, セ, ソ

- 213〜217: マ, ミ, ム, メ, モ

- 229〜233: ァ, ィ, ゥ, ェ, ォ
- 234: ッ
- 235〜237: ャ, ュ, ョ

### 5. 日本語記号領域 (238〜245)
